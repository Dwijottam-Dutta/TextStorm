
# TextStorm

A simple web text/notes editor with Mac UI/UX Design

#### Version 2.3.9 beta


## Top Features

- Light/dark mode toggle
- Full Screen for cool feel
- File Rename
- File Download and Save in Browser
- Text Manipulation
- MAC UI/UX design
- Editing/Viewing Mode feature for presentation
